# Comandos a utilizar!

``` 
npx sequelize-cli db:create  // Crea la base de datos.
```

```
npx sequelize-cli db:migrate   // Ejecuta las migraciones para crear tablas en BD.
```

```
npx sequelize-cli db:seed:all   // Inserta los seeders (datos de prueba) en la BD.
```

```
npm run dev     // Corre el archivo de ejecución en desarrollo (nodemon)
npm start       // Corre el archivo de ejecución en produccion (node)
npm test        // Se debería dejar libre para los test
```
